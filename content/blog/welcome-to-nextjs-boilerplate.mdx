# Welcome to Your Next.js Boilerplate

We're excited you've chosen our Next.js boilerplate for your next project! This isn't just another starter template – it's a **teachable** foundation designed to help you understand modern web development patterns while building secure, scalable applications.

## What Makes This Different

Most boilerplates show you *how to get something working*. This one shows you:

- **How to do it right** with security-first principles
- **Why decisions were made** through comprehensive documentation
- **Where things live** with clear, feature-based organization
- **How to extend safely** with patterns and best practices

## Tech Stack Highlights

### 🚀 Next.js 15.4.3 with App Router
Built on the latest Next.js with full App Router support, Server Components, and Server Actions for optimal performance.

### 🎨 Modern Styling
- **Tailwind CSS v4** (stable) for utility-first styling
- **shadcn/ui** components for beautiful, accessible UI
- **Dark mode** support out of the box

### 🔐 Supabase Integration
- **Authentication** with multiple providers
- **PostgreSQL** database with Row Level Security
- **Real-time subscriptions** for live features

### 🛡️ Security First
Every feature is built with security in mind:
- Input validation with Zod
- Row Level Security policies
- Proper authentication flows
- Security headers and CSP

## Perfect for Learning

Whether you're new to Next.js or want to understand modern patterns, this boilerplate includes:

- **Inline documentation** explaining every important file
- **Comprehensive guides** for each feature
- **ADRs** (Architecture Decision Records) explaining why we chose each tool
- **Best practices** embedded throughout
